# Soundscape

## Project Structure

- `index.html` - Vite app entry HTML.
- `src/main.tsx` - React bootstrap/root render.
- `src/pages/CarnaticWorkbenchPage.tsx` - Main app page and core UI/audio logic.
- `src/styles/carnatic-workbench.css` - Global/page styling.
- `package.json` - Scripts and dependencies.
- `tsconfig.json` / `vite.config.ts` - TypeScript and Vite config.
